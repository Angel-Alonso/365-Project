Authors: Angel Alonso, Jonathan Fitzgerald, Brandon Po, Jonathan Shuman

This is a group project, written for Lucas Pierce's CS365 Databases course at CalPoly. The core idea of the project is a API for tracking stock portfolios. It allows for users to simulate buying and selling stocks in a simulated playground setting to allow for users to learn the basics of how to trade stocks. The system will store all data in a relational database, and will offer endpoints for viewing and changing data based on what the users' needs.
